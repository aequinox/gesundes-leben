# Changelog

## [1.0.0] - 2025-04-02

### Added
- Initial release of the WordPress to Markdown Exporter
- TypeScript implementation with service-oriented architecture
- Support for Astro content collections
- Image downloading and processing
- Frontmatter generation
- Interactive configuration wizard
- Command-line interface

### Features
- Convert WordPress export XML to Markdown
- Download and save images (attached and scraped)
- Generate frontmatter for Astro content collections
- Support for year/month/post folder structures
- Support for custom post types
- Support for SSL configuration
- Support for image optimization
